# Bin2Dec & Dec2Bin

## Objective

This project serves as a study in fundamental web development concepts, focusing on:

- **Vanilla TypeScript**: Strong typing and Object-Oriented Programming (OOP) without framework abstractions.
- **CSS Architecture**: BEM methodology, CSS Variables, and responsive design using modern layout techniques (Grid/Flexbox).
- **No External UI Dependencies**: All components, validitions, and effects are built from scratch.

## Tech Stack

- **Core**: TypeScript, HTML5, CSS3.
- **Tooling**: Vite, ESLint, Prettier, Husky, Commitlint.
- **Deployment**: Vercel.

## Quick Start

1.  **Install dependencies**:

    ```bash
    npm install
    ```

2.  **Run development server**:

    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```
